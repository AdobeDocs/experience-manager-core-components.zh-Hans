# 在Adobe Experience Manager自适应Forms中启用和使用文本输入验证模式

## 概述

在Adobe Experience Manager (Adobe Experience Manager) Forms中，在模板策略级别控制文本输入组件的特定验证格式（如社会保险号、电子邮件地址或邮政编码）。 本指南介绍如何：

1. 访问模板编辑器，并配置文本输入组件策略以公开验证模式。
2. 创建自适应表单并将这些验证模式应用于文本输入组件。

## 先决条件

- 访问启用了Forms的Adobe Experience Manager实例。
- 编辑模板的权限（通常是`template-authors`组的一部分）。
- 创建和编辑自适应Forms的权限。

## 第1部分：在模板中启用验证模式

>[!VIDEO](https://video.tv.adobe.com/v/3480390/)

### 步骤1：访问“模板”控制台

1. 在Adobe Experience Manager主屏幕中，单击左侧边栏中的&#x200B;**工具**（锤子图标）。
2. 导航到&#x200B;**常规>模板**。
3. 选择包含项目模板（例如，核心组件示例）的文件夹。
4. 找到要修改的特定模板（例如，AF Blank）并将其选定。
5. 单击&#x200B;**编辑**（或直接打开模板）。

### 第2步：访问组件结构

1. 确保您处于&#x200B;**结构**&#x200B;视图中（如有必要，从右上角的模式下拉列表中选择“结构”）。
2. 找到主&#x200B;**布局容器**&#x200B;或允许使用文本输入组件的特定容器。
3. 单击容器以显示允许的组件列表。
4. 在列表中找到&#x200B;**自适应表单文本框**&#x200B;组件。
5. 单击组件名称旁边的&#x200B;**策略**&#x200B;图标（由文件/抽屉符号表示）。

### 步骤3：配置文本输入组件策略

1. 在策略编辑器中，单击窗口右侧的&#x200B;**格式**&#x200B;选项卡。
2. 您将看到可用验证模式的列表。 选中要启用的格式的复选框：
   - 电话号码
   - 社会保障号码
   - 电子邮件字母数字
   - 邮政编码

### 步骤4：定义并保存策略

1. 切换回&#x200B;**策略**&#x200B;选项卡（或检查左侧设置）。
2. 在&#x200B;**策略标题**&#x200B;字段中，输入清除的名称（例如，`new-textinput-default-policy`）。
3. 单击右上角的&#x200B;**完成**（复选标记）图标以保存更改。

## 第2部分：创建表单并使用验证模式

现在，模板中已启用验证模式，您可以创建自适应表单并将其应用于文本输入组件。

>[!VIDEO](https://video.tv.adobe.com/v/3480391/)

### 第1步：创建自适应表单

1. 导航到&#x200B;**Forms &amp; Documents**&#x200B;界面。
2. 单击右上角的&#x200B;**创建**&#x200B;按钮。
3. 从下拉选项中选择&#x200B;**自适应表单**。
4. 选择启用验证模式的模板（例如，**空白**），然后单击&#x200B;**下一步**。
5. 在Add Properties屏幕上：
   - 为您的表单输入&#x200B;**标题**（例如，“testPolicy”）。 **Name**&#x200B;字段会根据标题自动填充。
   - 确保选择了正确的&#x200B;**主题客户端库**（例如，`adaptiveform.theme.canvas3`）。
6. 单击&#x200B;**创建**。 出现一条成功消息。
7. 单击&#x200B;**编辑**&#x200B;以在编辑器中打开表单。

### 步骤2：添加文本输入组件

1. 在表单编辑器的左侧边栏中找到&#x200B;**组件**&#x200B;浏览器。
2. 使用搜索栏查找&#x200B;**自适应表单文本框**&#x200B;组件。 键入“text”会过滤列表。
3. 单击并将&#x200B;**自适应表单文本框**&#x200B;组件拖动到主画布区域。

### 步骤3：配置组件格式

1. 单击新添加的&#x200B;**文本输入**&#x200B;组件以将其选定。
2. 单击组件上方显示的工具栏中的&#x200B;**配置**&#x200B;图标（由扳手表示）。
3. 在属性对话框中，导航到&#x200B;**格式**&#x200B;选项卡。
4. 找到&#x200B;**类型**&#x200B;下拉菜单，并将选择更改为&#x200B;**电话号码**（或其他启用的格式）。

### 步骤4：配置数据验证

1. 当仍在组件配置对话框中，导航到&#x200B;**验证**&#x200B;选项卡。
2. 找到&#x200B;**验证模式**&#x200B;部分。
3. 单击&#x200B;**类型**&#x200B;下拉菜单。
4. 从验证模式列表中选择&#x200B;**电话号码**。 这可确保在用户输入的文本与有效的电话号码格式不匹配时表单生成错误。
5. 单击对话框右上角的&#x200B;**完成**（复选标记）图标以保存更改。

## 验证

要验证验证模式是否正常工作，请执行以下操作：

1. 在编辑器中预览表单。
2. 在&#x200B;**文本输入**&#x200B;组件中输入测试数据。
3. 确认：
   - 启用的验证模式显示在组件的“格式”和“验证”选项卡中。
   - 无效的输入触发了相应的错误消息。
   - 接受有效的输入，并且没有错误。

## 疑难解答

- **问题：**&#x200B;验证模式未出现在表单组件中。
   - **解决方案：**&#x200B;确保模板策略已正确保存，并且您在创建表单时使用正确的模板。

- **问题：**&#x200B;无法访问模板编辑器。
   - **解决方案：**&#x200B;验证您是否具有编辑模板（`template-authors`组中的成员资格）的必要权限。

- **问题：**&#x200B;文本输入组件未正确验证输入。
   - **解决方案：**&#x200B;在组件的“验证”选项卡中重新检查验证模式设置，并确保选择了正确的模式。

## 后续步骤

在Adobe Experience Manager自适应Forms中启用和使用文本输入验证模式后：

- 为其他数据类型配置其他验证模式。
- 探索其他组件策略以自定义表单行为。
- 设置表单提交处理和数据处理逻辑。
