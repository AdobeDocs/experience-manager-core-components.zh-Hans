# 在 Adobe Experience Manager 自适应表单中启用和使用文本输入验证模式

## 概述

在 Adobe Experience Manager (Adobe Experience Manager) Forms 中，文本输入组件的特定验证格式（如社会保障号码、电子邮件地址或邮政编码）在模板策略级别上进行控制。本指南介绍了如何：

1. 访问模板编辑器，配置文本输入组件策略，以公开验证模式。
2. 创建自适应表单，并将这些验证模式应用于文本输入组件。

## 先决条件

- 访问一个启用了 Forms 的 Adobe Experience Manager 实例。
- 编辑模板的权限（通常是 `template-authors` 组的一部分）。
- 创建和编辑自适应表单的权限。

## 第 1 部分：在模板中启用验证模式

>[!VIDEO](https://video.tv.adobe.com/v/3480390/)

### 步骤 1：访问模板控制台

1. 在 Adobe Experience Manager 主页屏幕中，点击左侧边栏中的&#x200B;**工具**（锤子图标）。
2. 导航到&#x200B;**一般 > 模板**。
3. 选择包含您的项目模板（例如核心组件示例）的文件夹。
4. 找到您想更改的特定模板（例如 AF 空白），将其选定。
5. 点击&#x200B;**编辑**（或直接打开模板）。

### 步骤 2：访问组件结构

1. 确保您在&#x200B;**结构**&#x200B;视图中（需要时，从右上角的模式下拉列表中选择“结构”）。
2. 找到主要&#x200B;**布局容器**&#x200B;或者允许使用您的文本输入组件的特定容器。
3. 点击容器，显示允许的组件列表。
4. 在列表中找到&#x200B;**自适应表单文本框**&#x200B;组件。
5. 点击组件名称旁边的&#x200B;**策略**&#x200B;图标（表示为文件/抽屉符号）。

### 步骤 3：配置文本输入组件策略

1. 在策略编辑器中，点击窗口右侧的&#x200B;**格式**&#x200B;选项卡。
2. 您将看到可用的验证模式的列表。勾选您想启用的格式的复选框：
   - 电话号码
   - 社会保障号码
   - 电子邮件字母数字
   - 邮政编码

### 步骤 4：定义并保存策略

1. 切换回到&#x200B;**策略**&#x200B;选项卡（或者检查左侧设置）。
2. 在&#x200B;**策略标题**&#x200B;字段中输入一个清晰的名称（例如 `new-textinput-default-policy`）。
3. 点击右上角的&#x200B;**完成**（对勾）图标，保存您做的更改。

## 第 2 部分：创建一个表单，并使用验证模式

现在，模板中已启用了验证模式，您可以创建一个自适应表单，并将这些模式应用于文本输入组件。

>[!VIDEO](https://video.tv.adobe.com/v/3480391/)

### 步骤 1：创建自适应表单

1. 导航到&#x200B;**表单和文档**&#x200B;界面。
2. 点击右上角的&#x200B;**创建**&#x200B;按钮。
3. 从下拉选项中选择 **自适应表单**。
4. 选择启用了验证模式的模板（例如 **空白**），然后点击&#x200B;**下一步**。
5. 在“添加属性”屏幕上：
   - 输入表单的&#x200B;**标题**（例如“testPolicy”）。**名称**&#x200B;字段会根据标题自动填充。
   - 确保选择了正确的&#x200B;**主题客户端库**（例如 `adaptiveform.theme.canvas3`）。
6. 单击&#x200B;**创建**。 现在会显示成功消息。
7. 点击&#x200B;**编辑**，在编辑器中打开表单。

### 步骤 2：添加一个文本输入组件

1. 在表单编辑器的左侧边栏中找到&#x200B;**组件**&#x200B;浏览器。
2. 使用搜索栏查找&#x200B;**自适应表单文本框**&#x200B;组件。输入“text”筛选列表。
3. 点击并将&#x200B;**自适应表单文本框**&#x200B;组件拖到主要画布区域上。

### 步骤 3：配置组件格式设置

1. 点击新添加的&#x200B;**文本输入**&#x200B;组件，将其选定。
2. 点击组件上方显示的工具栏中的&#x200B;**配置**&#x200B;图标（表示为扳手）。
3. 在属性对话框中，导航到&#x200B;**格式**&#x200B;选项卡。
4. 找到&#x200B;**类型**&#x200B;下拉菜单，改选为&#x200B;**电话号码**（或其他已启用的格式）。

### 步骤 4：配置数据验证

1. 仍在组件配置对话框中，导航到&#x200B;**验证**&#x200B;选项卡。
2. 找到&#x200B;**验证模式**&#x200B;部分。
3. 点击&#x200B;**类型**&#x200B;下拉菜单。
4. 从验证模式列表中选择&#x200B;**电话号码**。这可确保在用户输入的文本不是有效的电话号码格式时表单会生成错误。
5. 点击对话框右上角的&#x200B;**完成**（对勾）图标，保存您做的更改。

## 验证

要验证这些验证模式是否正常工作，请执行以下操作：

1. 在编辑器中预览表单。
2. 在&#x200B;**文本输入**&#x200B;组件中输入测试数据。
3. 确认：
   - 启用的验证模式显示在组件的“格式”和“验证”选项卡中。
   - 无效的输入会触发适当的错误消息。
   - 有效的输入被接受，没有错误。

## 疑难解答

- **问题：**&#x200B;验证模式未出现在表单组件中。
   - **解决方法：**&#x200B;确保模板策略已正确保存，并且您在创建表单时使用了正确的模板。

- **问题：**&#x200B;无法访问模板编辑器。
   - **解决方法：**&#x200B;验证您是否具有编辑模板的必要权限（是 `template-authors` 组中的成员）。

- **问题：**&#x200B;文本输入组件未正确验证输入。
   - **解决方法：**&#x200B;在组件的“验证”选项卡中重新检查验证模式设置，并确保选择了正确的模式。

## 后续步骤

在 Adobe Experience Manager 自适应表单中启用和使用文本输入验证模式后：

- 为其他数据类型配置额外的验证模式。
- 探索其他组件策略，以自定义表单行为。
- 设置表单提交处理和数据处理逻辑。
